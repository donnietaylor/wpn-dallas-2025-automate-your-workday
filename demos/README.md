# Demos

This folder contains live demonstration code and scripts used during the "Automate Your Workday with AI" session.

## Demo Structure

Each demo is organized in its own subdirectory with:
- Setup instructions
- Complete source code
- Sample data (where applicable)
- Requirements and dependencies

## Planned Demos

- **AI Email Assistant**: Automatically categorize and respond to emails using AI
- **Document Processor**: Extract and summarize information from documents
- **Task Automation**: Create intelligent workflows for common workplace tasks
- **Data Analysis**: AI-powered insights from workplace data
- **Meeting Assistant**: Automated meeting notes and action items

## Running the Demos

1. Navigate to the specific demo folder
2. Follow the setup instructions in each demo's README
3. Install required dependencies
4. Configure API keys and settings
5. Run the demo script

## Requirements

- Python 3.8+ or Node.js 16+
- API access to AI services (OpenAI, Azure AI, etc.)
- Various Python/JavaScript packages (listed in each demo)

## Support

If you encounter issues with any demo, please check the troubleshooting section in each demo's documentation or create an issue in this repository.
