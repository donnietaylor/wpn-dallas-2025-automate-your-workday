## What is n8n?

[n8n](https://n8n.io/) is a powerful workflow automation tool that allows you to connect various apps and services together. It's self-hostable, meaning you can run it on your own infrastructure for better control and privacy.

## Why Use n8n?

- **Visual Workflow Builder**: Create automation workflows using a visual interface
- **Self-Hosted**: Run on your own infrastructure for data privacy
- **Extensible**: Connect to hundreds of services and APIs
- **Open Source**: Free to use and modify
- **AI Integration**: Easily integrate with OpenAI and other AI services
