# Code Samples

Reusable code snippets and examples for AI automation projects.

## Structure

```
├── python/           # Python examples
│   └── ai_email_classifier.py
├── javascript/       # JavaScript examples
│   └── ai-text-summarizer.js
├── powershell/       # PowerShell examples
│   └── AI-MeetingScheduler.ps1
└── n8n/              # n8n workflow JSON files
    └── daily-handoff/
```

## Languages & Platforms

- **Python**: AI integration examples - [Browse](python/)
- **JavaScript/Node.js**: Web automation - [Browse](javascript/)
- **PowerShell**: Windows/cross-platform automation - [Browse](powershell/)
- **n8n**: Visual workflow automation - [Browse](n8n/)

## Usage

1. Browse the relevant folder
2. Copy the code to your project
3. Install dependencies
4. Configure and test

## PowerShell Notes

For PowerShell examples, use **PowerShell 7+** (`pwsh`) instead of Windows PowerShell 5.1 (`powershell`). See [requirements.psd1](powershell/requirements.psd1) for details.
