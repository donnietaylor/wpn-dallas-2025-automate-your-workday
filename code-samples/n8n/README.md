# n8n Workflows

This folder contains n8n workflow examples that you can import into your local n8n instance.

## Projects

### [Daily Handoff](daily-handoff/)

An automated workflow for daily team handoffs and status updates.

*Note: Workflow JSON files will be added to this folder after the project is completed.*

## How to Import Workflows

1. Start n8n locally (see [Installation Guide](../../docs/n8n-setup/installation.md))
2. Open n8n in your browser at `http://localhost:5678`
3. Go to Workflows menu
4. Click "Import from File"
5. Select the workflow JSON file from this folder
6. Configure credentials and settings as needed

## Contributing

To contribute n8n workflows:

1. Create your workflow in n8n
2. Test it thoroughly
3. Export as JSON (Download option in n8n)
4. Add to the appropriate project folder
5. Include a README explaining the workflow
